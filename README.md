# Rule-Based Chatbot
## Project Overview

This project implements a simple, conversational AI chatbot that responds to user inputs based on predefined rules. It utilizes natural language processing fundamentals, specifically pattern matching through regular expressions, to identify user intent and guide the conversation flow.

### Key Features
* **Pattern Matching:** Uses Python's `re` module to understand various user phrasings (e.g., greetings, questions about its identity, farewells).
* **Interactive Terminal UI:** Provides a seamless chat interface directly within the command line.
* **Lightweight & Fast:** Requires no external libraries or heavy machine learning models to run.

## Technologies Used

* **Language:** Python 3.x
* **Standard Libraries:** `re` (Regular Expressions)

## How to Run Locally

### 1. Requirements
Ensure you have Python installed on your system. This script relies entirely on Python's built-in libraries, so no additional installations are necessary.

### 2. Execute the Script
Run the Python file from your terminal:
```bash
python chatbot.py

```

### 3. Usage

Once the script is running, simply type your messages into the terminal and press Enter. The bot will respond based on its programmed rules. Type `bye`, `quit`, or `exit` to end the conversation.
